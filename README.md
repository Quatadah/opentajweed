# OpenTajweed

**An open-source, interactive web application designed to help users learn and master the rules of Tajweed for reciting the Holy Qur'an.**

OpenTajweed provides structured, easy-to-navigate guides based on classic Tajweed literature, complete with interactive quizzes, progress tracking, and a beautiful, modern interface.
